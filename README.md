## Install

### System Requirements
- Ruby `2.3.1` recommended
- Postgres 9.2+

### Fork and bundle
- Fork the project
- Run bundle to install gems
```bash
bundle install
```

### Database setup

[Download and Install Postgres](https://www.postgresql.org/download/)

- Create the databases:
```bash
rails db:create
```

- Run migrations to create tables:
```bash
rails db:migrate
```

### Running Tests

To run the tests, use:

```bash
bundle exec rspec
```
